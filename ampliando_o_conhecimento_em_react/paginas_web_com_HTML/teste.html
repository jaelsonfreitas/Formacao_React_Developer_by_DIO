O que são MetaTags

As meta tags em HTML são elementos usados para fornecer informações adicionais sobre uma página da web. Elas são colocadas no cabeçalho (head) do documento HTML e não são visíveis na página quando ela é exibida no navegador. Em vez disso, as meta tags fornecem metadados que descrevem características da página ou fornecem informações adicionais para os navegadores, motores de busca e outros serviços web.
