O que é CSS?

O CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos nas páginas web. O CSS separa o conteúdo da representação visual do site. Pense na decoracão da sua página. Utilizando o CSS é possivel alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos. Também pode criar tabelas, usar variações de layouts, ajustar imagens para suas respectivas telas e assim por diante.

O que é seletores.

Os seletores são usados para direcionar os elementos HTML em nossas páginas da web que queremos estilizar. Há uma grande variedade de seletores CSS disponiveis, permitindo uma precisão refinada ao selecionar os elementos a serem estilizados.

Exemplo:

Seletores de tipo:

h1 {
color:blue;
background-color: yellow;
}
p {
color: red;
}

Seletores de Classe:

.box { }

Seletores de Id:

#unique { }

Seletores de Atributos

a[title] { }

a[href="https://example.com" ] { }

Flexbox

O Flex box foi projetado tanto como um modelo de layout unidimensional quanto como um método capaz de organizar espacialmente os elementos em uma interface, alé de possuir capacidades avançadas de alinhamento. Quando se descreve o flexbox como sendo unidimensional, enfatiza-se o fato de que ele lida com o layot em uma dimensão de cada vez, seja uma linha ou uma coluna

Os eixos do flexbox

Para se utilizar o flexbox, é preciso ter em mente que todas as operações realizadas relacionam-se a dois eixos: o eixo principal e o eixo transversal. O eixo principal e definido através da propriedade flex-direction e o eixo transversal encontra-se na direção perpendicular a ele.

Pseudo-Classes e Elementos

Uma Pseudo-Classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado. Por exemplo, :hover pode ser usado para alterar a cor de um botão quando o usuário passar o cursor sobre ele.

Media Queries

Com a diversidade de dispositivos e tamanhos de tela disponíveis hoje, é crucial projetar páginas web que se adaptem a diferentes contextos. É aqui que as Media Queries em CSS entram em jogo. Elas permitem que você aplique estilos específicos com base nas características do dispositivo ou da tela.

O que são medias queries?

As Media Queries são uma parte fundamental do design responsivo na web. Elas permitem que você ajuste o estilo do seu site com base em características como largura de tela, altura de tela, orientação, resolução de tela, entre outras.

Sintaxe Básica A sintaxe básica de uma Media Query parece com isso:

  • `@media`: Indica que estamos usando uma Media Query.
  • `only screen`: Especifica que a regra se aplica apenas a dispositivos de tela (ignora impressoras, por exemplo).
  • `(max-width: 600px)`: A condição que deve ser atendida para aplicar os estilos dentro da Media Query.

Boas Práticas

  1. Pensamento Mobile-First:
    • Comece projetando para dispositivos móveis e adicione estilos à medida que a tela aumenta.
  2. Use `em` ou `rem` para Tamanhos de Fonte:
    • Isso ajuda a manter a consistência e facilita ajustes em várias telas.
  3. Teste em Vários Dispositivos:
    • Utilize ferramentas de desenvolvedor e teste em diferentes navegadores e dispositivos.

Conclusão

As Media Queries são uma ferramenta poderosa para criar layouts responsivos e garantir uma ótima experiência do usuário em uma ampla variedade de dispositivos. Ao aplicar essas técnicas, você pode melhorar a usabilidade e a acessibilidade do seu site, independentemente do dispositivo utilizado pelos visitantes.
